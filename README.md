3. Serverless
- 장점?
- 서버를 고려하지 않고 어플리케이션과 서비스를 구축하고 실행
- 비즈니스 로직에 집중할 수 있음
- 인프라 확장에 대한 고민이 줄어듬

- 단점?
- 아키텍쳐에 대한 이해가 필요함
- 디버깅하기가 어려움
- 가상 서버에 배포하는 것보다 비용이 더 나올 때도 존재함

- monolithic architecture vs microservice architecture
